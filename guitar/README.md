# Guitar

Tracking personal guitar skill progression. Intermediate player focused on building technique and expanding repertoire across genres.

## What's Here

- **songs.yaml** - Song database ported from Notion. Each entry includes difficulty, key, tuning, BPM, tab links, and practice notes. This is the source of truth going forward.
- **progress.yaml** - Self-assessment of skill areas (open chords, barre chords, fingerpicking, etc.) with 1-10 ratings and milestone tracking.
- **plan.md** - Practice plan, goals, and focus areas. Narrative format designed for AI agents to iterate on via PRs.
